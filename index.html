Advertising campaign – Sussex

This work focuses on a visual integration for advertising purposes, highlighting the absorption properties of Sussex paper towels. Through advanced use of photomontage and compositing techniques in Photoshop, a dramatic scene was created where a giant paper towel appears to block the flow of a river, leaving dry land behind it. The goal was to communicate the brand's value proposition using visual metaphor and realistic image manipulation.

Artistic integration – The Wheel of Time

This Photoshop composition explores a more artistic approach to integration, combining several elements from The Wheel of Time series. Techniques included color and lighting matching, custom brush strokes enhanced with fire filters to simulate floating magic, and multiple layered images that formed a fantasy-inspired background. The final artwork served as the base for a personal graphic seal overlay.

Integration – The Wheel of Time

This project involved the integration of brightly lit characters from The Wheel of Time into a dark hallway setting. It required precise work with light and shadow using tools like Burn, Dodge, Curves, Match Color, and projected shadows. The main goal was to achieve cohesion and realism, ensuring that all elements felt naturally placed within the same environment.

3D Illustration Digitally Traced in Illustrator

This illustration was developed through manual tracing and mesh gradient techniques to simulate three-dimensional volume and realistic lighting within a 2D vector environment. The Gradient Mesh tool allowed for smooth transitions of light and shadow across the object, giving it depth and dimension. The goal was to explore a more advanced use of Illustrator's tools to create a visually compelling and semi-realistic result.

Traced Image

This project involved creating a vector illustration based on a reference image for another artist, using the Image Trace tool and the Shape Builder tool (Shift + M) to merge and divide shapes. Custom strokes and brushes were added to enhance the visual richness and detail of the final piece. The focus was on developing a clean and technically coherent artwork, making the most of Illustrator’s non-destructive editing capabilities.

SKILLS

Save Maia – Interactive Game

"Save Maia" is an interactive narrative game that places players in a tense, immersive experience where every decision can mean life or death. The story follows Maia, a young woman kidnapped and trapped in a mysterious location. Players must choose how she reacts to different situations—attempting to escape, confront the captor, or seek help—uncovering new paths and consequences with each choice. This project combines design, storytelling, and user interaction to create a powerful digital experience.
